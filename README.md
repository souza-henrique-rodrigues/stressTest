Projeto com Docker, Express, PostgreSQL (Pagila) e Testes de Stress (Artillery)

Este repositório apresenta uma aplicação executada em ambiente conteinerizado, utilizando Docker e Docker Compose para orquestração, um servidor Express para processamento das requisições HTTP, um banco de dados PostgreSQL configurado para inicializar automaticamente com o sample Pagila, e a ferramenta Artillery para realização de testes de carga e stress.

Tecnologias Utilizadas

Docker – Empacotamento e isolamento dos serviços

Docker Compose – Orquestração e gerenciamento dos containers

Node.js / Express – Servidor HTTP e camada de aplicação

PostgreSQL + Pagila Sample – Banco de dados com dataset pré-carregado

Artillery – Ferramenta para testes de performance e stress
